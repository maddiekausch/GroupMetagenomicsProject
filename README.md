# Group Metagenomics Project -- Beatrice, Isabel, Sarah, Maddie K

This project aims to characterize viral species in a sample from a frozen bog (Bog frozen replicate A, accesion no. SAMN08784152) following the workflow from Trubl, et al. (2025). Our workflow begins with checking the quality of the reads, trimming low-quality reads using Trimmomatic, then checking quality again using FastQC. We then assembled a metagenome using the trimmed paired-end reads using MEGAHIT. We took the assembled contigs from our metagenome and used VirSorter2 for assembling the data into viral contigs, identifying 41. Using vclust, we clustered these viral contigs into vOTUs. We built a bowtie2 index of vOTUs from the class vOTU data, then mapped our reads onto this index using bowtie2 and samtools. Finally, we calculated richness, evenness, and abundance data, using R to generate diversity statistics and a heatmap of relative vOTU abundance in each of the class samples.
